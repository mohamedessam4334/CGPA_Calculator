# CGPA & GPA Calculator

A clean, responsive, and fully functional **CGPA (Cumulative GPA) Calculator** built with HTML, Tailwind CSS, and Vanilla JavaScript. No build steps, no dependencies — just open in a browser and start calculating.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Tech](https://img.shields.io/badge/stack-HTML%20%7C%20Tailwind%20%7C%20JS-orange)

## Features

- **Multi-Semester Support** — Add unlimited semesters and track your academic progress over time.
- **Real-Time Calculation** — GPA and CGPA update instantly as you type.
- **Weighted Average Formula** — `GPA = Σ(Grade Point × Credits) / Σ(Total Credits)`
- **Custom Grading Scale** — A+ (4.00), A (3.70), B+ (3.30), B (3.00), C+ (2.70), C (2.30), D+ (2.00), D (1.70), F (1.00)
- **Responsive Design** — Works seamlessly on desktop, tablet, and mobile.
- **Dynamic Course Rows** — Add or remove courses per semester with smooth animations.
- **Color-Coded Results** — GPA highlights change dynamically based on performance.

## How to Use

1. **Add Semesters** — Use the sidebar to add new semesters (e.g., Fall 2025, Spring 2026).
2. **Enter Courses** — For each semester, fill in the course name, credit hours, and grade.
3. **Watch It Calculate** — Semester GPA and overall CGPA update automatically.
4. **Share** — Send the link to friends to compare grades!

## Getting Started

### Option 1: Open Locally
Simply double-click the `index.html` file to open it in your browser.

### Option 2: Deploy to GitHub Pages
1. Fork or upload this repository to GitHub.
2. Go to **Settings → Pages**.
3. Select **Deploy from a branch** → `main` → `/ (root)`.
4. Your site will be live at `https://yourusername.github.io/cgpa-calculator`.

### Option 3: Deploy to Netlify
Drag and drop the `index.html` file onto [netlify.com/drop](https://netlify.com/drop) for instant hosting.
